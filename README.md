# 🖥️ PC Control Bot — Telegram бот для керування ПК

## 📋 Встановлення

### 1. Встанови Python
Завантаж з https://python.org (3.10+), обов'язково постав галочку "Add to PATH"

### 2. Встанови залежності
Відкрий папку з ботом у PowerShell і виконай:
```
pip install -r requirements.txt
```

### 3. Налаштуй bot.py
Відкрий `bot.py` і встав свої дані:
```python
BOT_TOKEN  = "токен_від_BotFather"   # @BotFather → /newbot
ALLOWED_ID = 123456789               # свій ID з @userinfobot
```

### 4. Запусти бота
```
python bot.py
```

## 🎮 Текстові команди

| Команда | Дія |
|---------|-----|
| `/start` | Головне меню з кнопками |
| `/help` | Список усіх команд |
| `write <текст>` | Показати текст у вікні на ПК |
| `open <програма>` | Відкрити програму |
| `url <посилання>` | Відкрити сайт у браузері |
| `aurl <посилання>` | Відкрити сайт в **інкогніто** |
| `search <запит>` | Пошук у Google |
| `amenu` | Швидке меню програм |
| `lock` | Заблокувати ПК (Win+L) |
| `theend` | Вимкнути ПК (через 10 сек) |

### Приклади
```
write Купи молоко!
open notepad.exe
open C:\Program Files (x86)\Steam\steam.exe
url youtube.com
aurl instagram.com
search погода Варшава
```

## 🔘 Кнопки головного меню

| Кнопка | Дія |
|--------|-----|
| 🔊 Гучніше | +10% гучності → показує поточний % |
| 🔉 Тихіше | -10% гучності → показує поточний % |

## 📋 Швидке меню (amenu)

Перші 2 — найважливіші, одразу доступні:

| Кнопка | Програма |
|--------|----------|
| 🎮 Steam | `C:\Program Files (x86)\Steam\steam.exe` |
| 💬 Discord | `AppData\Local\Discord\...` |

Далі: Блокнот, Провідник, Калькулятор, Медіаплеєр, Диспетчер, CMD, Chrome, PowerShell.

### Додати свою програму в amenu
У `bot.py` знайди список `AMENU_APPS` і додай рядок:
```python
("🎯 Назва", r"C:\повний\шлях\до\програми.exe"),
```
> **Де знайти шлях:** права кнопка на ярлик → Властивості → поле "Об'єкт"

## 🔊 Логіка гучності

Після кожної зміни гучності бот показує спливаюче повідомлення:
- `🔊 Гучність: 70%` — після підвищення
- `🔉 Гучність: 40%` — після зниження
- `🔇 Звук вимкнено` — якщо mute увімкнено
- `🔕 Гучність: 0% (тихо)` — після обнулення

Бот використовує **pycaw** для точного читання рівня гучності.
Якщо pycaw недоступний — автоматично перемикається на WinAPI (менш точний, але завжди працює).

## 🔁 Автозапуск при старті Windows

1. Натисни `Win + R`, введи `shell:startup`
2. Створи файл `start_bot.bat` з вмістом:
```bat
@echo off
pythonw "C:\шлях\до\bot.py"
```
3. Скопіюй `.bat` у папку автозапуску

---

## ⚠️ Важливо

- Бот працює **тільки на Windows**
- Яскравість працює тільки на **ноутбуках** або моніторах із WMI-підтримкою
- `aurl` відкриває Chrome у режимі інкогніто (потрібен Chrome)
- `theend` вимикає ПК через 10 секунд → скасувати: `shutdown /a` у CMD
- Перевір шляхи до TLauncher і Tor Browser у списку `AMENU_APPS`
