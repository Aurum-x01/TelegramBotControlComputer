# 🖥️ PC Control Bot — Telegram бот для керування ПК

## 📋 Встановлення

### 1. Встанови Python
Завантаж з https://python.org (3.10+), обов'язково постав галочку "Add to PATH"

### 2. Встанови залежності
Відкрий папку з ботом у PowerShell і виконай:
```
pip install -r requirements.txt
```

### 3. Отримай токен бота
- Напиши @BotFather у Telegram
- Введи `/newbot`, дай ім'я та username
- Скопіюй токен

### 4. Дізнайся свій Telegram ID
- Напиши боту @userinfobot
- Скопіюй своє `id` (число)

### 5. Налаштуй bot.py
Відкрий `bot.py` і заміни:
```python
BOT_TOKEN  = "YOUR_BOT_TOKEN_HERE"   # ← встав токен
ALLOWED_ID = 123456789               # ← встав свій user_id
```

### 6. Запусти бота
```
python bot.py
```

## 🎮 Команди

| Команда | Дія |
|---------|-----|
| `/start` | Головне меню з кнопками |
| `/help` | Список команд |
| `open notepad.exe` | Відкрити блокнот |
| `open chrome.exe` | Відкрити Chrome |
| `url google.com` | Відкрити сайт у браузері |
| `aurl youtube.com` | Відкрити сайт в **інкогніто** |
| `amenu` | Швидке меню програм |
| `lock` | Заблокувати ПК (Win+L) |
| `theend` | Вимкнути ПК (через 10 сек) |

## ⚙️ Додати свої програми в amenu

У файлі `bot.py` знайди список `AMENU_APPS` і додай свої рядки:
```python
AMENU_APPS = [
    ("🗒️ Блокнот",   "notepad.exe"),
    ("🎮 Steam",     r"C:\Program Files (x86)\Steam\steam.exe"),
    ("💬 Discord",   "discord.exe"),
    # додай сюди свої
]
```

## 🔁 Автозапуск при старті Windows

1. Натисни `Win + R`, введи `shell:startup`
2. Створи файл `start_bot.bat` з вмістом:
```bat
@echo off
pythonw "C:\шлях\до\bot.py"
```
3. Скопіюй `.bat` у папку автозапуску

---

## ⚠️ Важливо
- Бот працює **тільки на Windows**
- Яскравість працює тільки на **ноутбуках** або моніторах із WMI-підтримкою
- `aurl` відкриває Chrome у режимі інкогніто (потрібен Chrome)
- `theend` — вимикає ПК через 10 секунд (щоб скасувати: `shutdown /a`)
